# Indive-Tech-Solutions_Technical-Challenge

### Cara Menjalankan

- Jalankan `cp .env.example .env`
- Ubah credential .env dan config.json pada folder config
- Jalankan `npm install`
- Jalankan `npx sequelize-cli db:migrate`
- Jalankan `npm run dev`
